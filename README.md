# Applied Statistics Training (Industrial Context)

This repository contains materials from a 2-day training on applied statistics and data-driven decision-making.

## Objectives
- Understand descriptive statistics
- Model uncertainty (confidence intervals)
- Apply linear regression (OLS)
- Use statistical tests (Student, ANOVA, chi-square)

## Content
- 📊 Excel exercises
- 🐍 Python notebooks
- 📄 Slides (PDF)

## Teaching approach
- Hands-on sessions
- Real datasets
- Focus on interpretation and decision making
